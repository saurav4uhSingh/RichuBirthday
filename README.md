# Happy Birthday Richa 🎁✨

A highly polished, interactive, and beautifully designed birthday celebration web application crafted specifically for **Richu (Richa)**. 

This application offers an immersive, nostalgic college-best-friend vibe with responsive components, premium animations, and synchronous audio feedback.

---

## 🌟 Key Features

### 1. Interactive Cake Cutting Ceremony 🍰
* **Candle Lighting**: Use the virtual matches to light the candles with soft sound cue triggers.
* **Blow out Candles**: Blow over the screen or trigger the airflow buttons to see the candles extinguish with smoke trails and celebratory cascades.
* **Surgically Slice**: Cut the multi-tier decorated cake with a custom knife animation.
* **Feed the Slice**: Plate the slice and "feed" it virtual-style to trigger joyful hearts and a celebratory rain of treats.

### 2. Polaroid Memories Scrapbook 📸
* **Nostalgic Keepsakes**: Beautifully angled, realistic Polaroid cards with customized Hinglish comments recalling college memories, tea chats, canteen gossip, and supporting hands.
* **Custom Comments**: High-contrast, handwritten-style typography with custom positioning tags.
* **Local Persistence**: Save custom memory additions directly inside the browser's sandbox securely across refreshes.

### 3. Chronological Bestie Timeline 🎓
* **First Day Vali Pehchan**: The awkward first handshakes.
* **Constant Bakbak**: Chaotic chat histories and endless jokes.
* **Lectures Bunk & Chai**: Escaping tedious class sessions to the campus canteen.
* **Supporting Pillars**: Stands strong through study stress.
* **Dramebaaz Future**: Bright promises and hopes for the milestones ahead.

### 4. Surprise Box Portal 🎁
* Locked boxes containing warm encouragement quotes.
* A grand final secret lockbox that reveals a heartfelt, team-engraving birthday message with custom particles and visual chime cues.

---

## 🛠️ Built With

* **React 18 & Vite**: High-performance, lightning-fast rendering.
* **Tailwind CSS**: Elite custom color typography pairing and responsive layouts.
* **Framer Motion (`motion/react`)**: Cinematic micro-animations, bouncy entry elements, and fluid layouts.
* **Lucide React**: Clean, modern iconography.
* **Web Audio API**: Real-time synthesizer wave sound generators (wind blow gusts, cutting effects, chimes, pops) to guarantee complete native performance with zero external asset dependencies.

## 🚀 Local Development

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **Build Production Bundle**:
   ```bash
   npm run build
   ```

4. **Serve Production Build**:
   ```bash
   npm run start
   ```
